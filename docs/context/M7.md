## 이번 마일스톤 변경 파일

- `src/agents/qaAgent.ts` — 신규 생성. `runQA(devResult, spec, gameName): Promise<QAResult>` — manifest 무결성 검증(fileIntegrity), 전체 파일 내용 수집, QA 프롬프트 + API 호출, fileIntegrity=false 시 verdict 강제 REJECT
- `src/agents/plAgent.ts` — 수정. `evaluateQAResult(state, qaResult): PLDecision` — PASS→RELEASE, REJECT+canRetry→RETRY, REJECT+!canRetry→FAIL. `generateFailReport(state, reason): string` — 섹션 17 형식 FAIL 리포트
- `src/index.ts` — 수정. 전체 루프 완성: DEV_IMPLEMENT → QA_REVIEW → evaluateQAResult 판단 → RELEASE→DONE / RETRY→incrementRetry→DEV_IMPLEMENT 재진입 / FAIL→FAILED→generateFailReport 출력. QAResult 결과 출력 추가
