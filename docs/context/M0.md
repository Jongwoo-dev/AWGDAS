## 아키텍처 상태

### 동작하는 것
- `npm install` 성공
- `npm run build` → `dist/index.js` 생성
- `npm start` → 정상 종료
- `npm run typecheck` → 에러 0
- `npm run dev` → tsx로 src/index.ts 실행
- TypeScript strict mode + ESM (NodeNext) 빌드 파이프라인

### 아직 없는 것
- 공유 타입 정의 (`src/types/`)
- 유틸리티 함수 (`src/utils/`)
- 상태 머신 (`src/state/`)
- 에이전트 프롬프트 (`src/prompts/`)
- 에이전트 구현 (`src/agents/`)
- 테스트 설정 (vitest.config.ts)
- 실제 파이프라인 로직
